# qimen-graph 奇门节点图排盘

这是一个可直接部署到 GitHub Pages 的静态前端原型。目标不是直接给出神秘断语，而是像 `liuyao-graph` 一样，把奇门遁甲盘面的关键中间量拆解成：

> 时间起局 → 阴阳遁与局数 → 九宫门星神干 → 用神定位 → 关系边 → 状态修正 → 六维量化 → 结构化导出

## 当前特性

- 单文件 `index.html`，无需构建工具。
- 优先使用 `lunar-javascript` CDN 计算四柱、节气；不可用时回退内置近似算法。
- 默认采用时家奇门、转盘法、节气三元 / 符头定元。
- 支持时家、日家、月家、年家入口。
- 自动生成：
  - 阴遁 / 阳遁与局数
  - 年月日时四柱
  - 旬首、旬空、驿马
  - 地盘干、天盘干、暗干
  - 八门、九星、八神
  - 值符、值使
- 九宫卡片式展示，每宫显示：
  - 神、星、门、天干、地干、暗干
  - 空亡、马星、三奇、值符、值使、用神命中等标签
  - 六维量化：宫气、门用、星势、神助、格局、落地
